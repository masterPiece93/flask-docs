typechecksimplr :
This package provides :-
- @strict decorator
- TypeCheck class

@strict decorator :
applied on whichever python function , it checks for the type hints ( built-in types )

TypeCheck class :
helps in making simple Schema dataclass. A dataclass that inherits this class will check type hints
